# Henderson — Burgess-Stump Family Tree (spreadsheet)

Multi-sheet Excel workbook from the Henderson collection with four genealogical charts spanning the Burgess and Stump families from **Joseph Burgess** (b. 1735, Groby Lodge) through mid-20th-century descendants.

## Sheets

| Sheet | Scope |
|-------|-------|
| Burgess | Joseph Burgess → Henry → Edward → Fanny → Bottin children; George Burgess → American Burgess branch (Crane, Beinecke) |
| Burgess-Ransom | Ransom branch (connected via marriage) |
| Leicester Burgess | Leicester-area Burgess relations |
| George Burgess | George Burgess (b. 1817) branch in detail — Scarsdale NY descendants |

## Key data confirmed against vault

- Henriette Louise Fernande Bottin: b. 1888 Tehran (aligns with legacy export, not passport 1886)
- Alice Bottin: b. 1889 (widowed 1922, m. 1926 Carlo Marrobbio)
- Edward Bottin: b. 1891
- René Bottin: b. 1892
- George Bottin: b. 1895
- Étienne marriage: 1909, Tehran; b. 1880 Tallinn, d. 1951 Tehran
