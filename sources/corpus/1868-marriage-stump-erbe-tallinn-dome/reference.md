# Marriage — Markus Franz Stump and Olga Caroline Erbe, 22 June 1868, Tallinn

Three register pages from the Tallinn Dome Lutheran congregation (Toomkirik) record this marriage. The registers are in German.

| Field | Value |
|-------|-------|
| **Groom** | Markus Franz Stump |
| **Groom title** | Ober-Lehrer (senior teacher) |
| **Groom born** | 1834 (age 34) |
| **Bride** | Olga Caroline Erbe |
| **Bride born** | c. 1845 (age 23); EBC report gives 11 Nov 1844, St Petersburg |
| **Bride's father** | Hermann Erbe |
| **Date** | 22 June 1868 |
| **Place** | Reval (Tallinn), Estonia |
