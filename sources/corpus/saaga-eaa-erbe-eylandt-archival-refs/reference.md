# Estonian National Archives (Saaga) — Erbe / Eylandt / Busse / Schittmar archival references

Archival shelf marks cited on Geni profiles managed by **Elle Kiiker**, **Andrey Masing**, **Maris Lindau**, and **Lea Kornel**. Each reference points to a specific digitised page in the **Saaga** viewer (ra.ee/dgs). The images are publicly accessible but the viewer requires JavaScript — they cannot be fetched or scraped programmatically.

Status: **cited but not yet independently verified** against the scanned images. Each entry below records the archival reference, the Saaga URL with highlight coordinates, what the record is expected to show, and which parentage link it proves.

## Johann Jakob Eylandt — Kirbla family list / soul revision (1795)

| Field | Value |
|-------|-------|
| Archive | EAA (Estonian Historical Archives) |
| Fond / series / file | **EAA.1864.2.V-67** |
| Page | **227** |
| Saaga URL | `https://www.ra.ee/dgs/explorer.php?ark=EAA.1864.2.V-67&page=227` |
| Highlight coords | `1192,1353,1206,472,0` |
| Record type | Soul revision / family list (Hingeloend) |
| Parish | **Kirbla** (Märjamaa, Vigala kihelkond) |
| Date | **1795** |
| Expected content | Johann Jakob Eylandt (then aged ~21) listed as a member of the household of his father **Carl Gustav Eyland** — with mother **Margaretha Elisabeth** and siblings |
| Proves | **Johan Jacob ← Carl Gustav × Margarete Elisabeth** (I302 ← F143) |
| Geni source | Johann Jakob Eylandt profile, "About" section |

EAA.1864 is the fond for Kirbla parish. The "V-67" designation and the 1795 date indicate a Hingeloend (soul revision list) — a census-like household register compiled by the Lutheran pastor. These lists record heads of household with wives, children, ages, and sometimes occupations. If Johan Jacob appears as a son of Carl Gustav, this is a primary-source confirmation of that parentage.

This is the **only known primary-source citation** for the Carl Gustav → Johan Jacob parent-child link.
